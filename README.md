A full-featured, enterprise-grade HR Management System built with Java 17 & Spring Boot 3.  
Includes:
• Spring Boot webFlux reactive
• Role-based access (Admin, HR, Employee) with JWT  
• Async payroll processing via RabbitMQ  
• Redis caching for employee data  
• MySQL +  R2dbc  
• Swagger/OpenAPI documentation  
• Dockerized deployment (app + DB + Redis + RabbitMQ)  

Perfect for learning advanced Spring patterns or deploying in production.
